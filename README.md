# Employee Management System (EMS)

Simple demo app to showcase Java, Spring Boot, Hibernate (JPA), JSP, MySQL and REST APIs.

## Tech stack
- Java 17
- Spring Boot (Web, Data JPA, Security)
- Hibernate (via Spring Data JPA)
- JSP + JSTL
- Oracle
- Maven, JUnit 5, Mockito
